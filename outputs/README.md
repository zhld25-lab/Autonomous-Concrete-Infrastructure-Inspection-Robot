# Outputs

This folder stores files generated after running the notebooks. The included files are sample generated outputs from a successful run of the project.

Expected generated files include:

- `class_distribution.csv`
- `sample_images.png`
- `training_history.png`
- `confusion_matrix.png`
- `classification_report.txt`
- `prediction_examples.png`
- `crack_risk_map.png`
- `rl_training_rewards.png`
- `inspection_route.png`

These files can be regenerated by rerunning the notebooks after the dataset is placed under `data/raw/concrete_crack/`.
