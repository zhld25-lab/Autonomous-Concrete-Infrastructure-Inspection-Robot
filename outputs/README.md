# Outputs

This folder stores files generated after running the notebooks.

Expected generated files include:

- `class_distribution.csv`
- `sample_images.png`
- `training_history.png`
- `confusion_matrix.png`
- `classification_report.txt`
- `prediction_examples.png`
- `crack_risk_map.png`
- `rl_training_rewards.png`
- `inspection_route.png`

These files are intentionally not committed to GitHub because they are reproducible notebook outputs and may change after each run.
